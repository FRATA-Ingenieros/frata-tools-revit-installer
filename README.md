# Frata Tools para Revit

Frata Tools es un complemento para Autodesk Revit desarrollado por Frata Ingenieros SAC.

Este repositorio contiene el instalador oficial y las versiones disponibles del complemento.

## Version disponible

La version mas reciente disponible es:

| Version | Fecha de publicacion | Revit compatible | Instalador |
| --- | --- | --- | --- |
| `3.1.1` | 22/04/2026 | Revit 2022, 2023, 2024, 2025, 2026, 2027 | `FrataTools-3.1.1.exe` |

## Como instalar

1. Descargar el instalador de la version que corresponda.
2. Cerrar Autodesk Revit antes de iniciar la instalacion.
3. Ejecutar el archivo `FrataTools-3.1.1.exe`.
4. Seguir los pasos del asistente de instalacion.
5. Abrir Revit.
6. Verificar que Frata Tools aparezca en la cinta de herramientas de Revit.

## Recomendaciones antes de instalar

- Cerrar todas las sesiones abiertas de Revit.
- Usar una version de Revit compatible.
- Si Windows solicita permisos, permitir la ejecucion del instalador.
- Si ya existe una version anterior instalada, cerrar Revit antes de actualizar.

## Soporte

Si el complemento no aparece en Revit despues de instalarlo, verificar primero que la version de Revit sea compatible y que Revit haya estado cerrado durante la instalacion.
